Forknote Wallet 1.0.8.2 Release

  • November 25, 2015
  • pmitchev

Forknote has been updated to version 1.0.8.2 featuring various RPC methods added to the daemon:

  • f_get_blockchain_settings Returns the settings of the used configuration file.

  • f_blocks_list_json Returns list of shortly described blocks.

  • f_block_json Returns detailed description of a block.

  • f_transaction_json Returns detailed description of a transaction.

The new 1.0.8.2 version of Forknote is available in the downloads section.
